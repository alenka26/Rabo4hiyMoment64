# Rabo4hiyMoment64

Static business card mockup for the "Rabo4hiy Moment" concept.

## Preview

Open `index.html` in a browser to view the front and back of the card.

The layout is built with plain HTML and CSS:

- print-oriented 85mm x 55mm card proportions;
- responsive preview page for desktop and mobile screens;
- editable placeholder contact details;
- print styles for exporting the mockup from the browser.
